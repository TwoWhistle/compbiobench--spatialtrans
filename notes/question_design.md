# Benchmark Question Design Notes

## Source dataset

The questions use derived files from DLPFC 10x Visium sample 151673 from the spatialLIBD dataset. This dataset was used in Yuan et al. as part of the spatial clustering benchmark. The benchmark paper evaluates spatial clustering methods using prediction accuracy, spatial continuity, marker gene quality, scalability, robustness, and preprocessing/postprocessing effects.

## Question 1: srt-accuracy-q1

This question tests whether an agent can evaluate spatial clustering labels against ground-truth cortical layer annotations. It maps to the paper's prediction accuracy framework using normalized mutual information, homogeneity, and completeness.

The prediction columns are augmented method-like labels generated by perturbing the real cortical layer labels at different rates. This creates a controlled, objectively gradable benchmark question while preserving the real DLPFC label distribution.

## Question 2: srt-marker-q1

This question tests whether an agent can perform a simplified domain marker gene analysis and compute spatial autocorrelation using Moran's I.

It maps to the paper's marker gene evaluation framework, where spatial autocorrelation metrics are used to assess whether domain-specific marker genes have spatial structure.

## CompBioBench-style design principles

Each question is:
- self-contained
- file-based
- computational
- objectively gradable
- biologically motivated
- small enough to run in an isolated benchmark environment
